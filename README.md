## Development

1. Install [Python 3.11.4 amd64][https://www.python.org/downloads/].
1. Install packages.
    ```cmd
    pip install requests
    pip install opencv-python
    pip install python-decouple
    ```
1. Create copy `.env` file.
    ```cmd
    copy .env.example .env
    ```
1. Edit `.env` file.
1. Run python script.
    ```cmd
    python src/main.py
    ```
